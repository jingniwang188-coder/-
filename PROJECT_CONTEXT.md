# 塔罗之眼项目上下文

## 线上网站

- 正式地址：https://tarotwheel.vercel.app
- 支付确认后台：https://tarotwheel.vercel.app/admin.html
- 部署方式：GitHub 连接 Vercel，`main` 分支推送后由 Vercel 自动部署。
- 代码仓库远端：https://github.com/jingni0618/-.git

## Codex 协作约定

- 后续改动默认在本地完成校验后，由 Codex 直接提交并推送到 `main`。
- 推送成功后，无需用户再手动操作；等待 Vercel 自动构建完成即可看到线上更新。
- 系统生成文件如 `.DS_Store` 不纳入提交。

## 当前设计方向

- 视觉关键词：月光纸面、清晰墨色、少量宝石光。
- 体验关键词：少雾感、高对比、点击反馈明确、解读先给结论、档案能帮助复盘。
